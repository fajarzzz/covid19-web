# Covid19-web
Covid-19 Information Website, Build with NodeJS.

Consume API : https://github.com/mathdroid/covid-19-api

## RUN Web Application on Development
  Require : NPM NodeJS
  
  To run:
  ```
  npm install
  npm run start-dev
  ```
## RUN Web Application on Production 
  To run:
  ```
  cd dist/
  ```
  open **index.html**
